How to View This Slide Deck

  • This presentation is written entirely in SVG, HTML, CSS, and Javascript. Each slide is a standalone page, and they are bound together and ordered by a master list in a Javascript file referenced by each slide.
  • To move from slide to slide, press the right arrow button to move to the next slide, and the left arrow button to move to the previous slide.
  • To step through the bullet points on each slide, press the down arrow key to go forward, and the up arrow key to go backward.
  • (This slideshow should work in any browser, but text layout seems to be buggy in Safari. If you find other bugs, let me know.)
  • Please start the slideshow by pressing the right arrow key now. Enjoy!
